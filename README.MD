# Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project highlights my skills, projects, education, and experience while demonstrating responsive web design and interactive front-end development.

---

## Features

- Clean and modern user interface
- Fully responsive across desktop, tablet, and mobile devices
- Smooth scrolling navigation using JavaScript
- Interactive buttons and hover effects
- About Me section
- Skills section
- Featured Projects showcase
- Education & Internship section
- Contact section
- Dark theme design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## What I Learned

Building this project helped me improve my understanding of:

- Semantic HTML structure
- Responsive web design using Flexbox and Media Queries
- JavaScript DOM manipulation
- Smooth scrolling navigation
- Creating reusable and organized CSS
- Designing clean and user-friendly interfaces
